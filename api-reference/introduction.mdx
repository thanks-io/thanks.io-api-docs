## Welcome

The thanks.io API enables developers to integrate direct mail capabilities into their applications. With our API you can send any mailpiece you can imagine or add a recipient to an existing thanks.io automation.


## Authentication

### Bearer Token

All API endpoints are authenticated using Bearer tokens -  you can use it to authenticate your API requests by including it in the `Authorization` header as follows:

```http
Authorization: Bearer YOUR_ACCESS_TOKEN
```
